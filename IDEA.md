## 核心创意

### 概念

**球形脑空间 (Spherical Brain Space)** 是一种将真实神经元数据放入球形三维空间进行网络模拟的创新方法。

### 核心思想

1. **空间容器**：将神经元放置在一个球形的三维空间容器中
2. **真实数据**：使用真实的神经元形态数据（SWC格式），而非人工生成
3. **碰撞连接**：神经元通过物理碰撞自动形成突触连接
4. **自然演化**：神经元在空间中移动、连接、分裂、消亡，形成自然演化的网络

---

## 创新点

### 1. 球形空间约束
- 传统神经网络是平面或无约束的
- 本创意将神经元约束在球形空间内
- 球心区域营养充足，边缘区域营养稀少
- 形成自然的空间梯度

### 2. 真实神经元数据
- 使用FlyWire项目的真实神经元形态
- 保留真实的胞体位置、树突、轴突结构
- 基于LIF模型的真实电生理特性

### 3. 碰撞驱动连接
- 神经元在空间中随机移动
- 当两个神经元距离小于阈值时自动形成突触
- 模拟生物神经系统的自组织特性

### 4. 营养供给系统
- 空间中心营养浓度高
- 神经元根据饱腹度调整行为
- 饥饿时向中心移动，饱腹时向外扩散

### 5. 神经元生命周期
- 神经元可以分裂产生子体
- 子体在一定条件下会消亡
- 形成动态平衡的网络结构

## 技术实现

### 数据来源
- FlyWire - 果蝇全脑连接组项目
- FAFB (Full Adult Fly Brain) 数据集
- 神经元形态使用标准SWC格式

### 坐标转换
```
FAFB坐标（纳米）→ 球形空间坐标
X: 350000-550000 nm → -R ~ +R
Y: 80000-220000 nm → -R ~ +R
Z: 20000-240000 nm → -R ~ +R
```

### 神经元模型
- LIF (Leaky Integrate-and-Fire) 模型
- 静息电位: -70 mV
- 阈值电位: -55 mV
- 突触权重可调节
