## 二、核心设计决策

### 2.1 突触连接：碰撞自动触发

```
关键简化：神经元相遇 = 自动连接

规则：
├── 两个神经元距离 < 连接阈值 → 自动形成突触
├── 无需"生长突触"的决策过程
├── 每个神经元最多 3 个突触连接
└── 碰撞即连接，类似真实神经元的接触机制

实现逻辑：
if distance(neuron_a, neuron_b) < CONNECTION_THRESHOLD:
    if both have available synapse slots:
        create_synapse(neuron_a, neuron_b)  # 自动创建
```

### 2.2 神经元决策机制：随机驱动

```
决策内容：
├── 移动方向 → 随机 + 引力 + 饥饿驱动
├── 发送信号 → 随机生成
├── 是否分裂 → 条件触发 + 随机扰动
└── 突触连接 → 无需决策，碰撞自动触发！
```

### 2.3 营养供给系统（空间梯度）

```
营养供给规则（按距离中心点的位置）：

位置                    营养供给    说明
─────────────────────────────────────────────
外球边缘 (R)            3%         最外层
外球内部 (R/2 ~ R)      3%→10%     越靠近中球越多
中球表面 (R/2)          10%        引力终点
中球内部 (R/4 ~ R/2)    10%→15%    越靠近中心球越多
中心球表面 (R/4)        15%        最高供给点
中心球内部 (0 ~ R/4)    18%        核心区域

def get_nutrition_rate(distance, R):
    ratio = distance / R
    if ratio <= 0.25:
        return 0.18
    elif ratio <= 0.5:
        return 0.10 + (0.5 - ratio) / 0.25 * 0.05
    elif ratio <= 1.0:
        return 0.03 + (1.0 - ratio) / 0.5 * 0.07
    return 0.03
```

### 2.4 引力与旋转系统

```
引力系统：
├── 目的：像浮游生物一样，慢慢汇入中球表面
├── 方向：指向中心点
├── 强度：距离越远，引力越强
└── 终点：到达中球表面(R/2)后停止

中球旋转：
├── 方向：逆时针（从上往下看）
├── 速度：神经元移动速度的10%
├── 效果：带着神经元缓慢移动
└── 范围：只影响中球表面及内部的神经元
```

```python
class ConnectionSystem:
    """神经元连接系统 - 碰撞自动触发"""
    
    CONNECTION_DISTANCE = 20   # 连接距离阈值
    
    def check_and_connect(self, neurons):
        """检查碰撞并自动建立连接"""
        new_connections = []
        
        for i, n1 in enumerate(neurons):
            for n2 in neurons[i+1:]:
                # 计算距离
                distance = (n1.position - n2.position).magnitude()
                
                # 碰撞检测：距离足够近
                if distance < self.CONNECTION_DISTANCE:
                    # 检查是否已连接
                    if n2.id not in n1.synapses:
                        # 检查双方是否还有空位
                        if len(n1.synapses) < 3 and len(n2.synapses) < 3:
                            # 自动创建连接！
                            n1.synapses.append(n2.id)
                            n2.synapses.append(n1.id)
                            
                            # 重置未连接计数器
                            n1.steps_without_connection = 0
                            n2.steps_without_connection = 0
                            
                            new_connections.append((n1.id, n2.id))
        
        return new_connections
```

### 5.2 移动系统

```python
class MovementSystem:
    """神经元移动系统"""
    
    def __init__(self, R=1000):
        self.R = R
        self.base_speed = 1.0
        self.gravity_strength = 0.03
    
    def update_position(self, neuron, cluster_bonus=0.0):
        """更新神经元位置"""
        # 1. 随机移动方向
        random_dir = Vector3(
            random.uniform(-1, 1),
            random.uniform(-1, 1),
            random.uniform(-1, 1)
        ).normalized()
        
        direction = random_dir
        
        # 2. 引力影响（外球区域）
        distance = neuron.position.magnitude()
        if distance > self.R / 2:
            gravity_dir = -neuron.position.normalized()
            gravity_factor = self.gravity_strength * (distance / self.R)
            direction = direction + gravity_dir * gravity_factor
        
        # 3. 饥饿驱动（向营养高的区域）
        if neuron.satiety < 0.26:
            hunger_dir = -neuron.position.normalized()
            hunger_factor = (0.26 - neuron.satiety) * 2
            direction = direction + hunger_dir * hunger_factor
        
        # 4. 计算最终速度
        speed = self.base_speed * (1 + cluster_bonus)
        direction = direction.normalized()
        
        # 5. 更新位置
        neuron.position = neuron.position + direction * speed
        
        # 6. 中球旋转效应
        if distance <= self.R / 2 and distance > 0:
            rotation_speed = 0.1 / distance
            x, z = neuron.position.x, neuron.position.z
            neuron.position.x = x * math.cos(rotation_speed) - z * math.sin(rotation_speed)
            neuron.position.z = x * math.sin(rotation_speed) + z * math.cos(rotation_speed)
```

## 六、死亡系统详解

### 6.1 死亡条件

```
死亡触发条件：
├── 饥饿死亡：饱腹度 ≤ 0%
├── 孤立死亡：无连接超过存活时间（基础5000步）
```

### 6.2 死亡条件详解

| 死亡类型 | 触发条件 | 可变异影响 | 说明 |
|---------|---------|-----------|------|
| **饥饿死亡** | 饱腹度 ≤ 0% | 代谢速率影响消耗快慢 | 长期没吃到营养 |
| **孤立死亡** | 无连接 ≥ 存活时间 | 存活时间可变异 | 长期没有朋友 |

### 6.3 集群保护规则

```
集群保护机制：
├── 触发条件：集群神经元数量 ≥ 6
├── 保护效果：
│   ├── 不再分裂
│   └── 不再死亡
└── 目的：保护成熟集群稳定运行

示例：
├── 集群A有5个神经元 → 正常分裂/死亡
├── 集群B有6个神经元 → 受保护，不分裂不死亡
```

### 6.4 集群外交协商机制（≥12神经元）

```
外交触发条件：
├── 双方集群神经元数量都 ≥ 12
├── 双方集群边界神经元距离 < 连接阈值
└── 触发外交协商流程

协商流程：
├── 阶段1：接触（双方边界神经元靠近）
├── 阶段2：信息交互（交换集群信息）
├── 阶段3：决策（双方各自决定是否同意）
│   ├── 双方都同意 → 正式连接，合并为一个集群
│   └── 任一方不同意 → 分开，继续游荡
└── 阶段4：执行（连接或分开）
```

### 6.5 外交协商实现

```python
class DiplomacySystem:
    """集群外交协商系统"""
    
    DIPLOMACY_THRESHOLD = 12         # 外交触发阈值
    CONTACT_DISTANCE = 30            # 接触距离
    NEGOTIATION_STEPS = 50           # 协商所需步数
    
    def __init__(self):
        self.negotiations = {}       # 正在进行的协商
    
    def check_diplomacy(self, cluster_a, cluster_b, neurons, step):
        """检查是否触发外交"""
        
        if len(cluster_a['members']) < self.DIPLOMACY_THRESHOLD:
            return None
        if len(cluster_b['members']) < self.DIPLOMACY_THRESHOLD:
            return None
        
        border_a = self._get_border_neurons(cluster_a['members'], neurons)
        border_b = self._get_border_neurons(cluster_b['members'], neurons)
        
        min_distance = float('inf')
        for na in border_a:
            for nb in border_b:
                dist = na.position.distance_to(nb.position)
                min_distance = min(min_distance, dist)
        
        if min_distance > self.CONTACT_DISTANCE:
            return None
        
        negotiation_id = f"nego_{cluster_a['id']}_{cluster_b['id']}"
        
        if negotiation_id not in self.negotiations:
            self.negotiations[negotiation_id] = {
                'cluster_a': cluster_a['id'],
                'cluster_b': cluster_b['id'],
                'start_step': step,
                'phase': 'contact',
                'agreed_a': None,
                'agreed_b': None,
                'info_exchanged': False
            }
            print(f"[步骤{step}] 🤝 集群{cluster_a['id']}与{cluster_b['id']}开始接触")
        
        return self.negotiations[negotiation_id]
    
    def process_negotiation(self, negotiation, neurons, cluster_manager, step):
        """处理协商流程"""
        
        elapsed = step - negotiation['start_step']
        
        if negotiation['phase'] == 'contact' and elapsed >= 10:
            negotiation['phase'] = 'exchange'
            negotiation['info_exchanged'] = True
            print(f"[步骤{step}] 📡 集群开始信息交互...")
        
        if negotiation['phase'] == 'exchange' and elapsed >= 30:
            negotiation['phase'] = 'decision'
            negotiation['agreed_a'] = self._make_decision(negotiation['cluster_a'], neurons, cluster_manager)
            negotiation['agreed_b'] = self._make_decision(negotiation['cluster_b'], neurons, cluster_manager)
            print(f"[步骤{step}] 🗳️ 决策: 集群A={negotiation['agreed_a']}, 集群B={negotiation['agreed_b']}")
        
        if negotiation['phase'] == 'decision' and elapsed >= self.NEGOTIATION_STEPS:
            if negotiation['agreed_a'] and negotiation['agreed_b']:
                negotiation['phase'] = 'merge'
                print(f"[步骤{step}] ✅ 双方同意，开始合并！")
                return 'merge'
            else:
                negotiation['phase'] = 'separate'
                print(f"[步骤{step}] ❌ 协商失败，双方分开")
                return 'separate'
        
        return None
    
    def _make_decision(self, cluster_id, neurons, cluster_manager):
        """集群做出决策"""
        members = cluster_manager.clusters.get(cluster_id, set())
        
        total_satiety = 0
        count = 0
        for n in neurons:
            if n.id in members:
                total_satiety += n.satiety
                count += 1
        
        avg_satiety = total_satiety / count if count > 0 else 0.5
        
        agree_chance = 0.3 + avg_satiety * 0.5
        
        return random.random() < agree_chance
    
    def _get_border_neurons(self, member_ids, neurons):
        """获取边界神经元"""
        members = [n for n in neurons if n.id in member_ids]
        if not members:
            return []
        
        center = Vector3(0, 0, 0)
        for n in members:
            center = center + n.position
        center = center / len(members)
        
        border = []
        for n in members:
            if n.position.distance_to(center) > 50:
                border.append(n)
        
        return border if border else members[:3]
    
    def separate_clusters(self, cluster_a_id, cluster_b_id, cluster_manager, neurons):
        """分开两个集群"""
        members_a = cluster_manager.clusters.get(cluster_a_id, set())
        members_b = cluster_manager.clusters.get(cluster_b_id, set())
        
        for n in neurons:
            if n.id in members_a:
                n.velocity = n.velocity * -1
            elif n.id in members_b:
                n.velocity = n.velocity * -1
        
        print(f"  → 集群{cluster_a_id}和{cluster_b_id}开始反向移动")
```

```python
                'reason_cn': '孤立',
                'step': step,
                'neuron_id': neuron.id,
                'neuron_name': neuron.name,
                'isolation_steps': neuron.steps_without_connection,
                'survival_time': survival_time
            }
        
        return None

    def is_protected(neuron, cluster_manager) -> bool:
```

```python
class ClusterSystem:
    """集群管理系统"""
    
    def __init__(self):
        self.clusters = {}  # cluster_id -> set of neuron_ids
    
    def detect_clusters(self, neurons):
        """检测集群（BFS）"""
        visited = set()
        new_clusters = {}
        neuron_map = {n.id: n for n in neurons}
        
        for neuron in neurons:
            if neuron.id not in visited:
                cluster = self._bfs(neuron, neuron_map, visited)
                if len(cluster) >= 2:
                    cluster_id = f"cluster_{len(self.clusters) + len(new_clusters)}"
                    new_clusters[cluster_id] = cluster
        
        self.clusters.update(new_clusters)
        return new_clusters
    
    def _bfs(self, start, neuron_map, visited):
        cluster = set()
        queue = [start]
        
        while queue:
            neuron = queue.pop(0)
            if neuron.id in visited:
                continue
            
            visited.add(neuron.id)
            cluster.add(neuron.id)
            
            for connected_id in neuron.synapses:
                if connected_id not in visited:
                    connected = neuron_map.get(connected_id)
                    if connected:
                        queue.append(connected)
        
        return cluster
    
    def get_cluster_bonus(self, neuron_id):
        """获取神经元所属集群的速度加成"""
        for cluster_id, members in self.clusters.items():
            if neuron_id in members:
                size = len(members)
                if size == 2:
                    return 0.05
                elif size == 3:
                    return 0.10
                else:
                    return 0.10 + (size - 3) * 0.03
        return 0.0
```

```python
#!/usr/bin/env python3
"""
球形脑空间模拟器 v2.0
- 突触连接：碰撞自动触发
- 参考：FlyWire 连接组数据
"""

import random
import math
import time
from dataclasses import dataclass, field
from typing import List, Optional, Dict, Set

# ============ 数据结构 ============

@dataclass
class Vector3:
    x: float = 0.0
    y: float = 0.0
    z: float = 0.0
    
    def magnitude(self) -> float:
        return math.sqrt(self.x**2 + self.y**2 + self.z**2)
    
    def normalized(self) -> 'Vector3':
        m = self.magnitude()
        if m == 0:
            return Vector3(0, 0, 0)
        return Vector3(self.x/m, self.y/m, self.z/m)
    
    def __add__(self, other: 'Vector3') -> 'Vector3':
        return Vector3(self.x + other.x, self.y + other.y, self.z + other.z)
    
    def __sub__(self, other: 'Vector3') -> 'Vector3':
        return Vector3(self.x - other.x, self.y - other.y, self.z - other.z)
    
    def __mul__(self, scalar: float) -> 'Vector3':
        return Vector3(self.x * scalar, self.y * scalar, self.z * scalar)


@dataclass
class Neuron:
    id: str
    name: str
    position: Vector3
    velocity: Vector3 = field(default_factory=Vector3)
    satiety: float = 0.5
    synapses: List[str] = field(default_factory=list)
    cluster_id: Optional[str] = None
    state: str = 'idle'
    age: int = 0
    division_count: int = 0
    steps_without_connection: int = 0


# ============ 核心系统 ============

class BrainSpaceSimulator:
    """球形脑空间模拟器"""
    
    CONNECTION_DISTANCE = 20
    MAX_SYNAPSES = 3
    MAX_DIVISIONS = 99
    
    def __init__(self, R: int = 1000, initial_count: int = 33):
        self.R = R
        self.neurons: List[Neuron] = []
        self.clusters: Dict[str, Set[str]] = {}
        self.current_step = 0
        self.paused = False
        
        self._init_neurons(initial_count)
    
    def _init_neurons(self, count: int):
        names = ['小明', '小红', '小刚', '小芳', '小强', '小丽', '小军', '小美']
        for i in range(count):
            r = random.uniform(self.R * 0.25, self.R * 0.5)
            theta = random.uniform(0, 2 * math.pi)
            phi = random.uniform(0, math.pi)
            
            position = Vector3(
                r * math.sin(phi) * math.cos(theta),
                r * math.sin(phi) * math.sin(theta),
                r * math.cos(phi)
            )
            
            name = names[i % len(names)]
            if i >= len(names):
                name = f"{name}{i // len(names) + 1}"
            
            neuron = Neuron(
                id=f"neuron_{i+1:03d}",
                name=name,
                position=position
            )
            self.neurons.append(neuron)
    
    def step(self):
        if self.paused:
            return
        
        self.current_step += 1
        
        # 1. 更新集群
        self._update_clusters()
        
        # 2. 更新每个神经元
        for neuron in self.neurons:
            self._update_neuron(neuron)
        
        # 3. 碰撞检测 → 自动连接
        self._check_connections()
        
        # 4. 分裂检测
        self._check_divisions()
    
    def _update_neuron(self, neuron: Neuron):
        distance = neuron.position.magnitude()
        
        # 营养供给
        nutrition_rate = self._get_nutrition_rate(distance)
        neuron.satiety = min(1.0, neuron.satiety + nutrition_rate * 0.01)
        
        # 移动
        cluster_bonus = self._get_cluster_bonus(neuron.id)
        self._move_neuron(neuron, distance, cluster_bonus)
        
        # 更新状态
        neuron.age += 1
        if len(neuron.synapses) == 0:
            neuron.steps_without_connection += 1
    
    def _get_nutrition_rate(self, distance: float) -> float:
        ratio = distance / self.R
        if ratio <= 0.25:
            return 0.18
        elif ratio <= 0.5:
            return 0.10 + (0.5 - ratio) / 0.25 * 0.05
        elif ratio <= 1.0:
            return 0.03 + (1.0 - ratio) / 0.5 * 0.07
        return 0.03
    
    def _move_neuron(self, neuron: Neuron, distance: float, cluster_bonus: float):
        # 随机方向
        direction = Vector3(
            random.uniform(-1, 1),
            random.uniform(-1, 1),
            random.uniform(-1, 1)
        ).normalized()
        
        # 引力（外球区域）
        if distance > self.R / 2:
            gravity_dir = -neuron.position.normalized()
            gravity_factor = 0.03 * (distance / self.R)
            direction = direction + gravity_dir * gravity_factor
        
        # 饥饿驱动
        if neuron.satiety < 0.26:
            hunger_dir = -neuron.position.normalized()
            hunger_factor = (0.26 - neuron.satiety) * 2
            direction = direction + hunger_dir * hunger_factor
        
        # 计算速度
        speed = 1.0 * (1 + cluster_bonus)
        direction = direction.normalized()
        
        # 更新位置
        neuron.position = neuron.position + direction * speed
        
        # 中球旋转
        new_distance = neuron.position.magnitude()
        if new_distance <= self.R / 2 and new_distance > 0:
            rotation_speed = 0.1 / new_distance
            x, z = neuron.position.x, neuron.position.z
            neuron.position.x = x * math.cos(rotation_speed) - z * math.sin(rotation_speed)
            neuron.position.z = x * math.sin(rotation_speed) + z * math.cos(rotation_speed)
    
    def _check_connections(self):
        for i, n1 in enumerate(self.neurons):
            for n2 in self.neurons[i+1:]:
                distance = (n1.position - n2.position).magnitude()
                
                if distance < self.CONNECTION_DISTANCE:
                    if n2.id not in n1.synapses:
                        if len(n1.synapses) < self.MAX_SYNAPSES and len(n2.synapses) < self.MAX_SYNAPSES:
                            n1.synapses.append(n2.id)
                            n2.synapses.append(n1.id)
                            n1.steps_without_connection = 0
                            n2.steps_without_connection = 0
                            print(f"[步骤{self.current_step}] 💫 {n1.name} 与 {n2.name} 碰撞连接!")
    
    def _check_divisions(self):
        new_neurons = []
        for neuron in self.neurons:
            if neuron.division_count >= self.MAX_DIVISIONS:
                continue
            
            should_divide = False
            if neuron.satiety >= 1.0 and neuron.steps_without_connection >= 33:
                should_divide = random.random() < 0.5
            elif neuron.satiety >= 0.5 and neuron.steps_without_connection >= 66:
                should_divide = random.random() < 0.3
            
            if should_divide:
                child = Neuron(
                    id=f"{neuron.id}_d{neuron.division_count}",
                    name=f"{neuron.name}子",
                    position=neuron.position + Vector3(
                        random.uniform(-20, 20),
                        random.uniform(-20, 20),
                        random.uniform(-20, 20)
                    ),
                    satiety=0.5
                )
                new_neurons.append(child)
                neuron.division_count += 1
                neuron.satiety = 0.5
                print(f"[步骤{self.current_step}] 🔄 {neuron.name} 分裂产生 {child.name}!")
        
        self.neurons.extend(new_neurons)
    
    def _update_clusters(self):
        visited = set()
        self.clusters = {}
        
        for neuron in self.neurons:
            if neuron.id not in visited:
                cluster = self._bfs_cluster(neuron, visited)
                if len(cluster) >= 2:
                    cluster_id = f"cluster_{len(self.clusters)}"
                    self.clusters[cluster_id] = cluster
    
    def _bfs_cluster(self, start: Neuron, visited: Set[str]) -> Set[str]:
        cluster = set()
        queue = [start]
        neuron_map = {n.id: n for n in self.neurons}
        
        while queue:
            neuron = queue.pop(0)
            if neuron.id in visited:
                continue
            
            visited.add(neuron.id)
            cluster.add(neuron.id)
            
            for connected_id in neuron.synapses:
                if connected_id not in visited:
                    connected = neuron_map.get(connected_id)
                    if connected:
                        queue.append(connected)
        
        return cluster
    
    def _get_cluster_bonus(self, neuron_id: str) -> float:
        for cluster_id, members in self.clusters.items():
            if neuron_id in members:
                size = len(members)
                if size == 2:
                    return 0.05
                elif size == 3:
                    return 0.10
                else:
                    return 0.10 + (size - 3) * 0.03
        return 0.0
    
    def get_stats(self) -> dict:
        connected = sum(1 for n in self.neurons if len(n.synapses) > 0)
        avg_satiety = sum(n.satiety for n in self.neurons) / len(self.neurons) if self.neurons else 0
        total_connections = sum(len(n.synapses) for n in self.neurons) // 2
        
        return {
            'step': self.current_step,
            'neurons': len(self.neurons),
            'connected': connected,
            'connections': total_connections,
            'clusters': len(self.clusters),
            'avg_satiety': round(avg_satiety, 2)
        }
```

### 7.1 SWC 格式（标准神经元形态格式）

SWC 是 NeuroMorpho.Org 采用的标准格式，用于存储神经元三维结构：

```
SWC 格式说明（每行7个字段）：
├── 字段1: 节点ID（整数）
├── 字段2: 类型（0=未定义, 1=胞体, 2=轴突, 3=树突...）
├── 字段3: X坐标
├── 字段4: Y坐标
├── 字段5: Z坐标
├── 字段6: 半径
└── 字段7: 父节点ID（-1表示根节点）

示例 SWC 文件：
#n type x y z radius parent
1 1 0.0 0.0 0.0 10.0 -1
2 3 100.0 50.0 20.0 2.0 1
```

```python
#!/usr/bin/env python3
"""
球形脑空间模拟器 v2.1
- 突触连接：碰撞自动触发
- 数据存储：JSON + SWC格式
- 3D可视化：Matplotlib（低配置友好）

import random
import math
import time
import json
import os
from datetime import datetime
from dataclasses import dataclass, field
from typing import List, Optional, Dict, Set

# ============ 数据结构 ============

@dataclass
class Vector3:
    x: float = 0.0
    y: float = 0.0
    z: float = 0.0
    
    def magnitude(self) -> float:
        return math.sqrt(self.x**2 + self.y**2 + self.z**2)
    
    def normalized(self) -> 'Vector3':
        m = self.magnitude()
        if m == 0:
            return Vector3(0, 0, 0)
        return Vector3(self.x/m, self.y/m, self.z/m)
    
    def __add__(self, other: 'Vector3') -> 'Vector3':
        return Vector3(self.x + other.x, self.y + other.y, self.z + other.z)
    
    def __sub__(self, other: 'Vector3') -> 'Vector3':
        return Vector3(self.x - other.x, self.y - other.y, self.z - other.z)
    
    def __mul__(self, scalar: float) -> 'Vector3':
        return Vector3(self.x * scalar, self.y * scalar, self.z * scalar)


@dataclass
class Neuron:
    id: str
    name: str
    position: Vector3
    velocity: Vector3 = field(default_factory=Vector3)
    satiety: float = 0.5
    synapses: List[str] = field(default_factory=list)
    cluster_id: Optional[str] = None
    state: str = 'idle'
    age: int = 0
    division_count: int = 0
    steps_without_connection: int = 0


# ============ 存储系统 ============

class StorageSystem:
    """数据存储系统"""

        self.data_dir = data_dir
        self._ensure_dirs()

                {
                    "id": n.id,

# ============ 核心模拟器 ============

class BrainSpaceSimulator:
    """球形脑空间模拟器"""
    
    CONNECTION_DISTANCE = 20
    MAX_SYNAPSES = 3
    MAX_DIVISIONS = 99
    
    def __init__(self, R: int = 1000, initial_count: int = 33):
        self.R = R
        self.neurons: List[Neuron] = []
        self.clusters: Dict[str, Set[str]] = {}
        self.cluster_outputs: Dict[str, List[dict]] = {}
        self.current_step = 0
        self.paused = False
        self.storage = StorageSystem()
        
        self._init_neurons(initial_count)
    
    def _init_neurons(self, count: int):
        names = ['小明', '小红', '小刚', '小芳', '小强', '小丽', '小军', '小美']
        for i in range(count):
            r = random.uniform(self.R * 0.25, self.R * 0.5)
            theta = random.uniform(0, 2 * math.pi)
            phi = random.uniform(0, math.pi)
            
            position = Vector3(
                r * math.sin(phi) * math.cos(theta),
                r * math.sin(phi) * math.sin(theta),
                r * math.cos(phi)
            )
            
            name = names[i % len(names)]
            if i >= len(names):
                name = f"{name}{i // len(names) + 1}"
            
            neuron = Neuron(
                id=f"neuron_{i+1:03d}",
                name=name,
                position=position
            )
            self.neurons.append(neuron)
    
    def step(self):
        if self.paused:
            return
        
        self.current_step += 1
        
        self._update_clusters()
        
        for neuron in self.neurons:
            self._update_neuron(neuron)
        
        self._check_connections()
        self._check_divisions()
        
        self.storage.save_snapshot(self)
    
    def _update_neuron(self, neuron: Neuron):
        distance = neuron.position.magnitude()
        
        nutrition_rate = self._get_nutrition_rate(distance)
        neuron.satiety = min(1.0, neuron.satiety + nutrition_rate * 0.01)
        
        cluster_bonus = self._get_cluster_bonus(neuron.id)
        self._move_neuron(neuron, distance, cluster_bonus)
        
        neuron.age += 1
        if len(neuron.synapses) == 0:
            neuron.steps_without_connection += 1
    
    def _get_nutrition_rate(self, distance: float) -> float:
        ratio = distance / self.R
        if ratio <= 0.25:
            return 0.18
        elif ratio <= 0.5:
            return 0.10 + (0.5 - ratio) / 0.25 * 0.05
        elif ratio <= 1.0:
            return 0.03 + (1.0 - ratio) / 0.5 * 0.07
        return 0.03
    
    def _move_neuron(self, neuron: Neuron, distance: float, cluster_bonus: float):
        direction = Vector3(
            random.uniform(-1, 1),
            random.uniform(-1, 1),
            random.uniform(-1, 1)
        ).normalized()
        
        if distance > self.R / 2:
            gravity_dir = -neuron.position.normalized()
            gravity_factor = 0.03 * (distance / self.R)
            direction = direction + gravity_dir * gravity_factor
        
        if neuron.satiety < 0.26:
            hunger_dir = -neuron.position.normalized()
            hunger_factor = (0.26 - neuron.satiety) * 2
            direction = direction + hunger_dir * hunger_factor
        
        speed = 1.0 * (1 + cluster_bonus)
        direction = direction.normalized()
        
        neuron.position = neuron.position + direction * speed
        
        new_distance = neuron.position.magnitude()
        if new_distance <= self.R / 2 and new_distance > 0:
            rotation_speed = 0.1 / new_distance
            x, z = neuron.position.x, neuron.position.z
            neuron.position.x = x * math.cos(rotation_speed) - z * math.sin(rotation_speed)
            neuron.position.z = x * math.sin(rotation_speed) + z * math.cos(rotation_speed)
    
    def _check_connections(self):
        for i, n1 in enumerate(self.neurons):
            for n2 in self.neurons[i+1:]:
                distance = (n1.position - n2.position).magnitude()
                
                if distance < self.CONNECTION_DISTANCE:
                    if n2.id not in n1.synapses:
                        if len(n1.synapses) < self.MAX_SYNAPSES and len(n2.synapses) < self.MAX_SYNAPSES:
                            n1.synapses.append(n2.id)
                            n2.synapses.append(n1.id)
                            n1.steps_without_connection = 0
                            n2.steps_without_connection = 0
                            print(f"[步骤{self.current_step}] 💫 {n1.name} 与 {n2.name} 碰撞连接!")
    
    def _check_divisions(self):
        new_neurons = []
        for neuron in self.neurons:
            if neuron.division_count >= self.MAX_DIVISIONS:
                continue
            
            should_divide = False
            if neuron.satiety >= 1.0 and neuron.steps_without_connection >= 33:
                should_divide = random.random() < 0.5
            elif neuron.satiety >= 0.5 and neuron.steps_without_connection >= 66:
                should_divide = random.random() < 0.3
            
            if should_divide:
                child = Neuron(
                    id=f"{neuron.id}_d{neuron.division_count}",
                    name=f"{neuron.name}子",
                    position=neuron.position + Vector3(
                        random.uniform(-20, 20),
                        random.uniform(-20, 20),
                        random.uniform(-20, 20)
                    ),
                    satiety=0.5
                )
                new_neurons.append(child)
                neuron.division_count += 1
                neuron.satiety = 0.5
                print(f"[步骤{self.current_step}] 🔄 {neuron.name} 分裂产生 {child.name}!")
        
        self.neurons.extend(new_neurons)
    
    def _update_clusters(self):
        visited = set()
        old_clusters = self.clusters.copy()
        self.clusters = {}
        
        for neuron in self.neurons:
            if neuron.id not in visited:
                cluster = self._bfs_cluster(neuron, visited)
                if len(cluster) >= 2:
                    cluster_id = f"cluster_{len(self.clusters)}"
                    self.clusters[cluster_id] = cluster
                    
                    if cluster_id not in old_clusters:
                        self._on_new_cluster(cluster_id, cluster)
    
    def _on_new_cluster(self, cluster_id, members):
        """新集群形成时的处理"""
        print(f"[步骤{self.current_step}] 🌟 新集群形成: {cluster_id} ({len(members)}个神经元)")
        self.cluster_outputs[cluster_id] = []
        self.storage.save_cluster(cluster_id, members, [], self)
    
    def _bfs_cluster(self, start: Neuron, visited: Set[str]) -> Set[str]:
        cluster = set()
        queue = [start]
        neuron_map = {n.id: n for n in self.neurons}
        
        while queue:
            neuron = queue.pop(0)
            if neuron.id in visited:
                continue
            
            visited.add(neuron.id)
            cluster.add(neuron.id)
            
            for connected_id in neuron.synapses:
                if connected_id not in visited:
                    connected = neuron_map.get(connected_id)
                    if connected:
                        queue.append(connected)
        
        return cluster
    
    def _get_cluster_bonus(self, neuron_id: str) -> float:
        for cluster_id, members in self.clusters.items():
            if neuron_id in members:
                size = len(members)
                if size == 2:
                    return 0.05
                elif size == 3:
                    return 0.10
                else:
                    return 0.10 + (size - 3) * 0.03
        return 0.0
    
    def get_stats(self) -> dict:
        connected = sum(1 for n in self.neurons if len(n.synapses) > 0)
        avg_satiety = sum(n.satiety for n in self.neurons) / len(self.neurons) if self.neurons else 0
        total_connections = sum(len(n.synapses) for n in self.neurons) // 2
        
        return {
            'step': self.current_step,
            'neurons': len(self.neurons),
            'connected': connected,
            'connections': total_connections,
            'clusters': len(self.clusters),
            'avg_satiety': round(avg_satiety, 2)
        }

        self.ax = self.fig.add_subplot(111, projection='3d')
    
    def draw(self):
        self.ax.clear()
        
        R = self.brain.R
        self._draw_sphere_wireframe(R, 'blue', 0.1)
        self._draw_sphere_wireframe(R/2, 'green', 0.15)
        self._draw_sphere_wireframe(R/4, 'red', 0.2)
        
        for neuron in self.brain.neurons:
            x, y, z = neuron.position.x, neuron.position.y, neuron.position.z
            color = plt.cm.RdYlGn(neuron.satiety)
            size = 30 + len(neuron.synapses) * 20
            self.ax.scatter(x, y, z, c=[color], s=size, alpha=0.8)
        
        drawn = set()
        for neuron in self.brain.neurons:
            for target_id in neuron.synapses:
                edge = tuple(sorted([neuron.id, target_id]))
                if edge not in drawn:
                    drawn.add(edge)
                    target = next((n for n in self.brain.neurons if n.id == target_id), None)
                    if target:
                        self.ax.plot(
                            [neuron.position.x, target.position.x],
                            [neuron.position.y, target.position.y],
                            [neuron.position.z, target.position.z],
                            'gray', alpha=0.3, linewidth=0.5
                        )
        
        self.ax.set_xlabel('X')
        self.ax.set_ylabel('Y')
        self.ax.set_zlabel('Z')
        self.ax.set_title(f'球形脑空间 - 步骤 {self.brain.current_step} | 神经元: {len(self.brain.neurons)}')
        
        plt.pause(0.01)
    
    def _draw_sphere_wireframe(self, radius, color, alpha):
        u = np.linspace(0, 2 * np.pi, 15)
        v = np.linspace(0, np.pi, 8)
        x = radius * np.outer(np.cos(u), np.sin(v))
        y = radius * np.outer(np.sin(u), np.sin(v))
        z = radius * np.outer(np.ones(np.size(u)), np.cos(v))
        self.ax.plot_wireframe(x, y, z, color=color, alpha=alpha, linewidth=0.3)
```

```
移除 vs 死亡 的区别：

┌─────────────┬─────────────────┬─────────────────┐
│             │      死亡        │      移除        │
├─────────────┼─────────────────┼─────────────────┤
│ 触发方式    │ 自动（饥饿/孤立） │ 人工干预         │
│ 数据保留    │ 记录死亡原因     │ 完整保存到文件   │

│ 文件输出    │ 无               │ 自动导出BCF      │
│ 显示状态    │ 💀 死亡标记      │ 📤 移除标记      │
│ 集群影响    │ 可能导致集群解散 │ 集群整体移除     │
└─────────────┴─────────────────┴─────────────────┘

移除场景：
├── 单神经元表现不佳 → 移除，不再培养
├── 集群发展不理想 → 整体移除
├── 需要调整实验 → 移除部分神经元
└── 腾出空间给新神经元 → 移除旧的
```

### 10.3 移除系统实现

```python
import os
from datetime import datetime

class RemovalSystem:
    """移除系统 - 从培养皿取出神经元/集群"""
    
    REMOVED_DIR = "brain_space_data/removed"
    
    def __init__(self, brain):
        self.brain = brain
        self.removed_items = []  # 移除历史
        self.removal_log = []    # 移除日志
        
        os.makedirs(self.REMOVED_DIR, exist_ok=True)
    
    def remove_neuron(self, neuron_id: str, reason: str = "人工移除") -> dict:
        """移除单个神经元"""
        
        neuron = self._find_neuron(neuron_id)
        if not neuron:
            return {'success': False, 'error': '神经元不存在'}
        
        if neuron.id in self.brain.cluster_manager.neuron_to_cluster:
            cluster_id = self.brain.cluster_manager.neuron_to_cluster[neuron.id]
            return self.remove_cluster(cluster_id, reason)
        
        filename = self._export_removed_neuron(neuron)
        
        self.brain.neurons.remove(neuron)
        
        for n in self.brain.neurons:
            if neuron_id in n.synapses:
                n.synapses.remove(neuron_id)
        
        removal_record = {
            'type': 'neuron',
            'id': neuron_id,
            'name': neuron.name,
            'reason': reason,
            'step': self.brain.current_step,
            'timestamp': datetime.now().isoformat(),
            'filename': filename
        }
        
        self.removed_items.append(removal_record)
        self._log_removal(removal_record)
        
        print(f"[步骤{self.brain.current_step}] 📤 移除神经元: {neuron.name}")
        print(f"   原因: {reason}")
        print(f"   已保存到: {filename}")
        
        return {
            'success': True,
            'removal': removal_record
        }
    
    def remove_cluster(self, cluster_id: str, reason: str = "人工移除") -> dict:
        """移除整个集群"""
        
        if cluster_id not in self.brain.cluster_manager.clusters:
            return {'success': False, 'error': '集群不存在'}
        
        members = self.brain.cluster_manager.clusters[cluster_id].copy()
        
        cluster_neurons = [n for n in self.brain.neurons if n.id in members]
        
        filename = self._export_removed_cluster(cluster_id, cluster_neurons)
        
        for neuron in cluster_neurons:
            self.brain.neurons.remove(neuron)
        
        del self.brain.cluster_manager.clusters[cluster_id]
        
        for nid in members:
            if nid in self.brain.cluster_manager.neuron_to_cluster:
                del self.brain.cluster_manager.neuron_to_cluster[nid]
        
        for n in self.brain.neurons:
            for nid in members:
                if nid in n.synapses:
                    n.synapses.remove(nid)
        
        removal_record = {
            'type': 'cluster',
            'id': cluster_id,
            'member_count': len(members),
            'member_names': [n.name for n in cluster_neurons],
            'reason': reason,
            'step': self.brain.current_step,
            'timestamp': datetime.now().isoformat(),
            'filename': filename
        }
        
        self.removed_items.append(removal_record)
        self._log_removal(removal_record)
        
        print(f"[步骤{self.brain.current_step}] 📤 移除集群: {cluster_id}")
        print(f"   包含神经元: {len(members)}个")
        print(f"   原因: {reason}")
        print(f"   已保存到: {filename}")
        
        return {
            'success': True,
            'removal': removal_record
        }
    
    def restore_last(self) -> dict:
        """恢复最近移除的项目"""
        
        if not self.removed_items:
            return {'success': False, 'error': '没有可恢复的项目'}
        
        last_removal = self.removed_items.pop()
        
        if last_removal['type'] == 'neuron':
            return self._restore_neuron(last_removal)
        else:
            return self._restore_cluster(last_removal)
    
    def _restore_neuron(self, removal_record) -> dict:
        """恢复单个神经元"""
        
        filename = removal_record['filename']
        if not os.path.exists(filename):
            return {'success': False, 'error': '文件不存在'}
        
        data = self.brain.storage.load_bcf(filename)
        
        neuron_data = data['neurons'][0]
        neuron = Neuron(
            id=neuron_data['id'],
            name=neuron_data['name'],
            position=Vector3(*neuron_data['position']),
            satiety=neuron_data['satiety'],
            synapses=neuron_data.get('synapses', []),
            age=neuron_data.get('age', 0),
            division_count=neuron_data.get('division_count', 0)
        )
        
        self.brain.neurons.append(neuron)
        
        print(f"[步骤{self.brain.current_step}] 📥 恢复神经元: {neuron.name}")
        
        return {'success': True, 'restored': removal_record}
    
    def _restore_cluster(self, removal_record) -> dict:
        """恢复集群"""
        
        filename = removal_record['filename']
        if not os.path.exists(filename):
            return {'success': False, 'error': '文件不存在'}
        
        data = self.brain.storage.load_bcf(filename)
        
        for neuron_data in data['neurons']:
            neuron = Neuron(
                id=neuron_data['id'],
                name=neuron_data['name'],
                position=Vector3(*neuron_data['position']),
                satiety=neuron_data['satiety'],
                synapses=neuron_data.get('synapses', []),
                age=neuron_data.get('age', 0),
                division_count=neuron_data.get('division_count', 0)
            )
            self.brain.neurons.append(neuron)
        
        self.brain.cluster_manager.update_clusters(self.brain.neurons)
        
        print(f"[步骤{self.brain.current_step}] 📥 恢复集群: {len(data['neurons'])}个神经元")
        
        return {'success': True, 'restored': removal_record}
    
    def _export_removed_neuron(self, neuron) -> str:
        """导出被移除的神经元"""
        timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
        filename = os.path.join(
            self.REMOVED_DIR, 
            f"removed_{neuron.id}_{timestamp}.bcf"
        )
        
        self.brain.storage.save_bcf(
            f"removed_{neuron.id}",
            [neuron],
            [],
            {'reason': 'removed', 'original_id': neuron.id},
            filename
        )
        
        return filename
    
    def _export_removed_cluster(self, cluster_id, neurons) -> str:
        """导出被移除的集群"""
        timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
        filename = os.path.join(
            self.REMOVED_DIR,
            f"removed_{cluster_id}_{timestamp}.bcf"
        )
        
        connections = []
        for n in neurons:
            for syn_id in n.synapses:
                if any(syn.id == syn_id for syn in neurons):
                    connections.append((n.id, syn_id))
        
        self.brain.storage.save_bcf(
            cluster_id,
            neurons,
            connections,
            {'reason': 'removed', 'original_cluster_id': cluster_id},
            filename
        )
        
        return filename
    
    def _log_removal(self, record):
        """记录移除日志"""
        self.removal_log.append(record)
        
        log_file = os.path.join(self.REMOVED_DIR, "removal_log.json")
        
        import json
        logs = []
        if os.path.exists(log_file):
            with open(log_file, 'r', encoding='utf-8') as f:
                logs = json.load(f)
        
        logs.append(record)
        
        with open(log_file, 'w', encoding='utf-8') as f:
            json.dump(logs, f, ensure_ascii=False, indent=2)
    
    def _find_neuron(self, neuron_id):
        for n in self.brain.neurons:
            if n.id == neuron_id:
                return n
        return None
    
    def get_removal_history(self, limit: int = 10) -> list:
        """获取移除历史"""
        return self.removed_items[-limit:]
```

```python
def show_removal_history(removal_system, limit=5):
    """显示移除历史"""
    history = removal_system.get_removal_history(limit)
    
    if not history:
        print("暂无移除记录")
        return
    
    print("\n┌─────────────────────────────────────────────────────────────┐")
    print("│  移除历史                                                    │")
    print("├─────────────────────────────────────────────────────────────┤")
    
    for i, record in enumerate(reversed(history), 1):
```

### 11.1 集群定义与状态

```
集群 = 通过突触连接的神经元网络（≥2个神经元）

状态统计示例：
├── 总神经元: 33个
├── 单神经元: 10个（未连接）

│   ├── 集群1: 3个神经元
│   └── 集群2: 20个神经元
```

### 11.2 集群合并规则

```
合并触发：
├── 单神经元遇到单神经元 → 连接 → 形成新集群（2个）
├── 单神经元遇到集群 → 连接 → 加入集群
├── 集群遇到集群 → 连接 → 合并为更大集群
└── 连接后自动重新计算所有集群

合并流程：
┌─────────────────────────────────────────────────────────────┐
│  状态: 10个单神经元 + 集群A(3个) + 集群B(20个)              │
│                                                             │
│  集群A的神经元与集群B的神经元碰撞连接                        │
│                          ↓                                  │
│  合并检测: 两个集群有连接 → 合并                             │
│                          ↓                                  │
│  新状态: 10个单神经元 + 集群C(23个)                          │
│                                                             │
│  集群C = 集群A + 集群B + 新连接                              │
└─────────────────────────────────────────────────────────────┘
```

### 11.3 集群合并实现

```python
class ClusterManager:
    """集群管理器"""
    
    def __init__(self):
        self.clusters = {}  # cluster_id -> set of neuron_ids
        self.neuron_to_cluster = {}  # neuron_id -> cluster_id
    
    def update_clusters(self, neurons):
        """更新集群状态（每次连接后调用）"""
        visited = set()
        new_clusters = {}
        
        for neuron in neurons:
            if neuron.id not in visited:
                cluster = self._bfs_cluster(neuron, neurons, visited)
                if len(cluster) >= 2:
                    cluster_id = f"cluster_{len(new_clusters)}"
                    new_clusters[cluster_id] = cluster
        
        old_cluster_ids = set(self.clusters.keys())
        new_cluster_ids = set(new_clusters.keys())
        
        merged = old_cluster_ids != new_cluster_ids
        
        self.clusters = new_clusters
        self.neuron_to_cluster = {}
        for cid, members in new_clusters.items():
            for nid in members:
                self.neuron_to_cluster[nid] = cid
        
        return merged
    
    def get_stats(self, total_neurons):
        """获取统计信息"""
        clustered = sum(len(members) for members in self.clusters.values())
        single = total_neurons - clustered
        
        return {
            'total_neurons': total_neurons,
            'single_neurons': single,
            'cluster_count': len(self.clusters),
            'clusters': [
                {'id': cid, 'size': len(members)}
                for cid, members in sorted(self.clusters.items(), key=lambda x: -len(x[1]))
            ]
        }
    
    def _bfs_cluster(self, start, neurons, visited):
        """BFS查找连接的神经元"""
        cluster = set()
        queue = [start]
        neuron_map = {n.id: n for n in neurons}
        
        while queue:
            neuron = queue.pop(0)
            if neuron.id in visited:
                continue
            
            visited.add(neuron.id)
            cluster.add(neuron.id)
            
            for connected_id in neuron.synapses:
                if connected_id not in visited:
                    connected = neuron_map.get(connected_id)
                    if connected:
                        queue.append(connected)
        
        return cluster
```

```python
class BrainSpaceAPI:
    """球形脑空间 API"""
    
    def __init__(self, brain):
        self.brain = brain
        self.cluster_manager = brain.cluster_manager
    
    def send_input(self, target_id, content):
        """发送输入到神经元或集群"""
        if target_id in self.cluster_manager.neuron_to_cluster:
            cluster_id = self.cluster_manager.neuron_to_cluster[target_id]
            return self._send_to_cluster(cluster_id, content)
        else:
            return self._send_to_neuron(target_id, content)
    
    def _send_to_neuron(self, neuron_id, content):
        """发送输入到单个神经元"""
        neuron = self._get_neuron(neuron_id)
        if not neuron:
            return {'success': False, 'error': 'neuron not found'}
        
        neuron.add_input(content, self.brain.current_step)
        self.brain.message_log.log_input(neuron_id, content, self.brain.current_step)
        
        return {
            'success': True,
            'target_type': 'neuron',
            'target_id': neuron_id,
            'target_name': neuron.name
        }
    
    def _send_to_cluster(self, cluster_id, content):
        """发送输入到集群"""
        if cluster_id not in self.cluster_manager.clusters:
            return {'success': False, 'error': 'cluster not found'}
        
        members = self.cluster_manager.clusters[cluster_id]
        
        self.brain.message_log.log_input(cluster_id, content, self.brain.current_step)
        
        for neuron_id in members:
            neuron = self._get_neuron(neuron_id)
            if neuron:
                neuron.add_input(f"[集群输入] {content}", self.brain.current_step)
        
        return {
            'success': True,
            'target_type': 'cluster',
            'target_id': cluster_id,
            'member_count': len(members)
        }
    
    def get_output(self, source_id):
        """获取神经元或集群的输出"""
        if source_id.startswith('cluster_'):
            return self._get_cluster_output(source_id)
        else:
            return self._get_neuron_output(source_id)
    
    def _get_neuron_output(self, neuron_id):
        """获取单个神经元的输出"""
        neuron = self._get_neuron(neuron_id)
        if not neuron:
            return {'success': False, 'error': 'neuron not found'}
        
        outputs = neuron.output_history[-10:]
        
        return {
            'success': True,
            'source_type': 'neuron',
            'source_id': neuron_id,
            'source_name': neuron.name,
            'outputs': outputs
        }
    
    def _get_cluster_output(self, cluster_id):
        """获取集群的输出"""
        if cluster_id not in self.cluster_manager.clusters:
            return {'success': False, 'error': 'cluster not found'}
        
        outputs = self.brain.message_log.get_cluster_output(cluster_id)
        
        return {
            'success': True,
            'source_type': 'cluster',
            'source_id': cluster_id,
            'member_count': len(self.cluster_manager.clusters[cluster_id]),
            'outputs': outputs
        }
    
    def list_targets(self):
        """列出所有可通信的目标"""
        single_neurons = [
            {'id': n.id, 'name': n.name, 'type': 'neuron'}
            for n in self.brain.neurons
            if n.id not in self.cluster_manager.neuron_to_cluster
        ]
        
        clusters = [
            {
                'id': cid,
                'type': 'cluster',
                'member_count': len(members),
                'members': list(members)
            }
            for cid, members in self.cluster_manager.clusters.items()
        ]
        
        return {
            'single_neurons': single_neurons,
            'clusters': clusters,
            'total_neurons': len(self.brain.neurons)
        }
    
    def _get_neuron(self, neuron_id):
        for n in self.brain.neurons:
            if n.id == neuron_id:
                return n
        return None
```

## 十三、集群饱腹度系统

### 13.1 饱腹度计算规则

```
基础规则：
├── 单神经元基础饱腹度上限: 100%
├── 每增加1个神经元: +60% 饱腹度上限
├── 超过9个神经元后: 每增加1个只 +30%
└── 神经元数量上限: 无限

计算公式：
if cluster_size <= 9:
    max_satiety = 100% + (cluster_size - 1) × 60%
else:
    max_satiety = 100% + 8 × 60% + (cluster_size - 9) × 30%
```

### 13.3 饱腹度实现

```python
class SatietySystem:
    """饱腹度系统"""
    
    BASE_SATIETY = 100      # 基础饱腹度上限
    BONUS_PER_NEURON = 60   # 每个神经元增加的上限
    BONUS_AFTER_9 = 30      # 超过9个后的增加量
    THRESHOLD_9 = 9         # 9个神经元的阈值
    
    @staticmethod
    def calculate_max_satiety(cluster_size):
        """计算集群最大饱腹度"""
        if cluster_size <= 1:
            return SatietySystem.BASE_SATIETY
        
        if cluster_size <= SatietySystem.THRESHOLD_9:
            return SatietySystem.BASE_SATIETY + (cluster_size - 1) * SatietySystem.BONUS_PER_NEURON
        else:
            base_up_to_9 = SatietySystem.BASE_SATIETY + 8 * SatietySystem.BONUS_PER_NEURON
            extra = (cluster_size - SatietySystem.THRESHOLD_9) * SatietySystem.BONUS_AFTER_9
            return base_up_to_9 + extra
    
    @staticmethod
    def calculate_cluster_satiety(neurons, cluster_members):
        """计算集群当前饱腹度"""
        total = 0
        count = 0
        for n in neurons:
            if n.id in cluster_members:
                total += n.satiety
                count += 1
        
        if count == 0:
            return 0, 0
        
        current = total / count * 100
        max_satiety = SatietySystem.calculate_max_satiety(count)
        
        return current, max_satiety
    
    @staticmethod
    def get_satiety_percentage(current, max_satiety):
        """获取饱腹度百分比"""
        if max_satiety == 0:
            return 0
        return min(100, current / max_satiety * 100)
```

```python
@dataclass
class MutationAttributes:
    """变异属性"""
    
    synapse_distance: float = 20.0      # 突触连接距离
    speed_multiplier: float = 1.0       # 速度倍率
    max_satiety: float = 100.0          # 饭量上限（%）
    max_connections: int = 3            # 最大连接数
    perception_range: float = 50.0      # 感知范围
    metabolism_rate: float = 1.0        # 代谢速率
    division_threshold: float = 0.9     # 分裂阈值
    survival_time: int = 5000           # 无连接存活时间
    
    mutation_count: int = 0             # 变异次数
    mutation_history: List[dict] = field(default_factory=list)


class MutationSystem:
    """变异系统"""
    
    MUTATION_CHANCE_ON_DIVISION = 0.10   # 分裂时变异概率
    MUTATION_CHANCE_LONG_SURVIVAL = 0.05 # 长期存活变异概率
    MUTATION_CHANCE_MERGE = 0.03         # 合并时变异概率
    MUTATION_CHANCE_RANDOM = 0.01        # 随机变异概率
    
    POSITIVE_WEIGHT = 0.60               # 正向变异权重
    NEGATIVE_WEIGHT = 0.30               # 负向变异权重
    NEUTRAL_WEIGHT = 0.10                # 中性变异权重
    
    @staticmethod
    def try_mutate(neuron, trigger: str, step: int) -> Optional[dict]:
        """尝试变异"""
        
        chances = {
            'division': MutationSystem.MUTATION_CHANCE_ON_DIVISION,
            'long_survival': MutationSystem.MUTATION_CHANCE_LONG_SURVIVAL,
            'merge': MutationSystem.MUTATION_CHANCE_MERGE,
            'random': MutationSystem.MUTATION_CHANCE_RANDOM
        }
        
        if random.random() > chances.get(trigger, 0):
            return None
        
        mutation_type = MutationSystem._choose_mutation_type()
        mutation = MutationSystem._apply_mutation(neuron, mutation_type, step)
        
        if mutation:
            neuron.mutations.mutation_count += 1
            neuron.mutations.mutation_history.append(mutation)
        
        return mutation
    
    @staticmethod
    def _choose_mutation_type() -> str:
        """选择变异类型"""
        r = random.random()
        if r < MutationSystem.POSITIVE_WEIGHT:
            return 'positive'
        elif r < MutationSystem.POSITIVE_WEIGHT + MutationSystem.NEGATIVE_WEIGHT:
            return 'negative'
        else:
            return 'neutral'
    
    @staticmethod
    def _apply_mutation(neuron, mutation_type: str, step: int) -> Optional[dict]:
        """应用变异"""
        attrs = neuron.mutations
        
        mutation_options = [
            ('synapse_distance', 15.0, 40.0, 5.0),
            ('speed_multiplier', 0.5, 2.0, 0.1),
            ('max_satiety', 80.0, 150.0, 10.0),
            ('max_connections', 2, 5, 1),
            ('perception_range', 30.0, 100.0, 10.0),
            ('metabolism_rate', 0.5, 1.5, 0.1),
            ('division_threshold', 0.8, 0.95, 0.05),
            ('survival_time', 3000, 10000, 500),
        ]
        
        attr_name, min_val, max_val, change = random.choice(mutation_options)
        current_val = getattr(attrs, attr_name)
        
        if mutation_type == 'positive':
            if attr_name in ['metabolism_rate', 'division_threshold']:
                new_val = max(min_val, current_val - change)
            else:
                new_val = min(max_val, current_val + change)
        elif mutation_type == 'negative':
            if attr_name in ['metabolism_rate', 'division_threshold']:
                new_val = min(max_val, current_val + change)
            else:
                new_val = max(min_val, current_val - change)
        else:
            new_val = current_val + random.uniform(-change, change)
            new_val = max(min_val, min(max_val, new_val))
        
        if isinstance(current_val, int):
            new_val = int(round(new_val))
        
        setattr(attrs, attr_name, new_val)
        
        return {
            'step': step,
            'type': mutation_type,
            'attribute': attr_name,
            'old_value': current_val,
            'new_value': new_val,
            'trigger': '分裂' if trigger == 'division' else 
                       '长期存活' if trigger == 'long_survival' else
                       '集群合并' if trigger == 'merge' else '随机'
        }
```

### 17.1 触发条件

```
自动添加规则：
├── 单神经元数量 < 3 → 提示用户添加
├── 用户长时间未操作 → 程序自动添加
└── 目标：保持至少3个单神经元

判断逻辑：
single_neurons = total_neurons - clustered_neurons
if single_neurons < 3:

    if 超过30秒未操作:

```

### 17.2 实现代码

```python
import time

class AutoAddSystem:
    """自动添加神经元系统"""
    
    MIN_SINGLE_NEURONS = 3
    PROMPT_DELAY = 10      # 提示后等待秒数
    AUTO_ADD_DELAY = 30    # 自动添加等待秒数
    
    def __init__(self, brain):
        self.brain = brain
        self.prompt_time = None
        self.auto_add_time = None
        self.prompted = False
    
    def check_and_prompt(self):
        """检查并提示/自动添加"""
        single_count = self._count_single_neurons()
        
        if single_count >= self.MIN_SINGLE_NEURONS:
            self.prompted = False
            self.prompt_time = None
            self.auto_add_time = None
            return None
        
        now = time.time()
        
        if not self.prompted:
            self.prompted = True
            self.prompt_time = now
            self.auto_add_time = now + self.AUTO_ADD_DELAY
            return {
                'action': 'prompt',
                'message': f'单神经元数量不足({single_count}<3)，请添加神经元',
                'auto_add_in': self.AUTO_ADD_DELAY
            }
        
        if now >= self.auto_add_time:
            self._auto_add_neuron()
            self.auto_add_time = now + self.AUTO_ADD_DELAY
            return {
                'action': 'auto_add',
                'message': '自动添加了1个神经元'
            }
        
        remaining = int(self.auto_add_time - now)
        return {
            'action': 'waiting',
            'message': f'{remaining}秒后自动添加神经元'
        }
    
    def _count_single_neurons(self):
        """计算单神经元数量"""
        clustered = set()
        for members in self.brain.clusters.values():
            clustered.update(members)
        
        return len(self.brain.neurons) - len(clustered)
    
    def _auto_add_neuron(self):
        """自动添加神经元"""
        names = ['新', '生', '命', '星', '光', '灵', '智', '慧']
        idx = len(self.brain.neurons)
        name = names[idx % len(names)]
        if idx >= len(names):
            name = f"{name}{idx // len(names) + 1}"
        
        r = random.uniform(self.brain.R * 0.25, self.brain.R * 0.5)
        theta = random.uniform(0, 2 * math.pi)
        phi = random.uniform(0, math.pi)
        
        neuron = Neuron(
            id=f"neuron_{idx+1:03d}",
            name=name,
            position=Vector3(
                r * math.sin(phi) * math.cos(theta),
                r * math.sin(phi) * math.sin(theta),
                r * math.cos(phi)
            ),
            satiety=0.5
        )
        self.brain.neurons.append(neuron)
        print(f"[自动添加] ✨ 新神经元: {name}")
```

### 13.3 BCF 文件实现

```python
import struct
import json

BCF_MAGIC = b'BCF1'
BCF_VERSION = 1

class BCFFormat:
    """Brain Cluster File 格式"""
    
    @staticmethod
    def save(cluster_id, neurons, connections, metadata, filename):
        """保存为BCF文件"""
        
        neuron_data = [
            {
                'id': n.id,
                'name': n.name,
                'position': [n.position.x, n.position.y, n.position.z],
                'satiety': n.satiety,
                'synapses': n.synapses,
                'age': n.age,
                'division_count': n.division_count,
                'input_history': n.input_history[-100:],
                'output_history': n.output_history[-100:]
            }
            for n in neurons
        ]
        
        metadata_json = json.dumps(metadata, ensure_ascii=False)
        neuron_json = json.dumps(neuron_data, ensure_ascii=False)
        
        connection_binary = BCFFormat._encode_connections(connections)
        
        header = struct.pack(
            '<4sIIIQQ',
            BCF_MAGIC,
            BCF_VERSION,
            len(neurons),
            len(metadata_json.encode('utf-8')),
            len(neuron_json.encode('utf-8')) + len(connection_binary)
        )
        
        with open(filename, 'wb') as f:
            f.write(header)
            f.write(metadata_json.encode('utf-8'))
            f.write(neuron_json.encode('utf-8'))
            f.write(connection_binary)
        
        return filename
    
    @staticmethod
    def load(filename):
        """加载BCF文件"""
        with open(filename, 'rb') as f:
            header = f.read(32)
            magic, version, neuron_count, metadata_size, data_size = struct.unpack(
                '<4sIIIQQ', header
            )
            
            if magic != BCF_MAGIC:
                raise ValueError("Invalid BCF file")
            
            metadata_json = f.read(metadata_size).decode('utf-8')
            metadata = json.loads(metadata_json)
            
            remaining = f.read()
            
            try:
                split_point = remaining.rfind(b']') + 1
                neuron_json = remaining[:split_point].decode('utf-8')
                connection_binary = remaining[split_point:]
                
                neurons = json.loads(neuron_json)
                connections = BCFFormat._decode_connections(connection_binary)
            except:
                neurons = json.loads(remaining.decode('utf-8'))
                connections = []
            
            return {
                'metadata': metadata,
                'neurons': neurons,
                'connections': connections
            }
    
    @staticmethod
    def _encode_connections(connections):
        """编码连接数据"""
        data = struct.pack('<I', len(connections))
        for conn in connections:
            from_bytes = conn['from'].encode('utf-8')
            to_bytes = conn['to'].encode('utf-8')
            data += struct.pack('<H', len(from_bytes))
            data += from_bytes
            data += struct.pack('<H', len(to_bytes))
            data += to_bytes
            data += struct.pack('<If', conn.get('since', 0), conn.get('strength', 1.0))
        return data
    
    @staticmethod
    def _decode_connections(data):
        """解码连接数据"""
        if len(data) < 4:
            return []
        
        count = struct.unpack('<I', data[:4])[0]
        connections = []
        offset = 4
        
        for _ in range(count):
            from_len = struct.unpack('<H', data[offset:offset+2])[0]
            offset += 2
            from_id = data[offset:offset+from_len].decode('utf-8')
            offset += from_len
            
            to_len = struct.unpack('<H', data[offset:offset+2])[0]
            offset += 2
            to_id = data[offset:offset+to_len].decode('utf-8')
            offset += to_len
            
            since, strength = struct.unpack('<If', data[offset:offset+8])
            offset += 8
            
            connections.append({
                'from': from_id,
                'to': to_id,
                'since': since,
                'strength': strength
            })
        
        return connections


def export_cluster_to_bcf(cluster_id, cluster_members, all_neurons, message_log, step):
    """导出集群为BCF文件"""
    neuron_map = {n.id: n for n in all_neurons}
    cluster_neurons = [neuron_map[nid] for nid in cluster_members if nid in neuron_map]
    
    connections = []
    for n in cluster_neurons:
        for syn_id in n.synapses:
            if syn_id in cluster_members:
                connections.append({
                    'from': n.id,
                    'to': syn_id,
                    'since': step,
                    'strength': 1.0
                })
    
    metadata = {
        'cluster_id': cluster_id,
        'created_at': datetime.now().isoformat(),
        'neuron_count': len(cluster_neurons),
        'connection_count': len(connections) // 2,
        'export_step': step,
        'source': 'brain_space_simulator',
        'tags': [],
        'description': f'导出于步骤 {step}'
    }
    
    filename = f"cluster_{cluster_id}_{datetime.now().strftime('%Y%m%d_%H%M%S')}.bcf"
    return BCFFormat.save(cluster_id, cluster_neurons, connections, metadata, filename)


def import_cluster_from_bcf(filename, brain):
    """从BCF文件导入集群"""
    data = BCFFormat.load(filename)
    
    imported_neurons = []
    for n_data in data['neurons']:
        neuron = Neuron(
            id=n_data['id'],
            name=n_data['name'],
            position=Vector3(*n_data['position']),
            satiety=n_data['satiety'],
            synapses=n_data['synapses'],
            age=n_data.get('age', 0),
            division_count=n_data.get('division_count', 0)
        )
        neuron.input_history = n_data.get('input_history', [])
        neuron.output_history = n_data.get('output_history', [])
        imported_neurons.append(neuron)
    
    brain.neurons.extend(imported_neurons)
    
    return data['metadata']
```

## 二十四、球空间大小动态调整

### 24.1 调整功能说明

```
目的：
├── 缩小空间 → 加快神经元相遇连接
├── 放大空间 → 给神经元更多活动空间
└── 动态调整 → 根据实验需要灵活控制

快捷键：
├── [ 键 → 缩小空间（半径 -10%）
├── ] 键 → 放大空间（半径 +10%）
├── { 键 → 快速缩小（半径 -50%）
└── } 键 → 快速放大（半径 +100%）
```

### 24.2 空间调整实现

```python
class SpaceAdjuster:
    """球空间大小调整器"""
    
    MIN_RADIUS = 200    # 最小半径
    MAX_RADIUS = 5000   # 最大半径
    DEFAULT_RADIUS = 1000
    
    def __init__(self, brain):
        self.brain = brain
        self.R = self.DEFAULT_RADIUS
        self.adjustment_history = []
    
    def shrink(self, factor=0.9):
        """缩小空间"""
        old_R = self.R
        self.R = max(self.MIN_RADIUS, self.R * factor)
        
        if self.R != old_R:
            self._adjust_neuron_positions(old_R, self.R)
            self._record_adjustment('shrink', old_R, self.R)
            print(f"[空间调整] 缩小: {old_R:.0f} → {self.R:.0f}")
        
        return self.R
    
    def expand(self, factor=1.1):
        """放大空间"""
        old_R = self.R
        self.R = min(self.MAX_RADIUS, self.R * factor)
        
        if self.R != old_R:
            self._adjust_neuron_positions(old_R, self.R)
            self._record_adjustment('expand', old_R, self.R)
            print(f"[空间调整] 放大: {old_R:.0f} → {self.R:.0f}")
        
        return self.R
    
    def quick_shrink(self):
        """快速缩小（50%）"""
        return self.shrink(0.5)
    
    def quick_expand(self):
        """快速放大（100%）"""
        return self.expand(2.0)
    
    def reset(self):
        """重置为默认大小"""
        old_R = self.R
        self.R = self.DEFAULT_RADIUS
        self._adjust_neuron_positions(old_R, self.R)
        print(f"[空间调整] 重置: {old_R:.0f} → {self.R:.0f}")
        return self.R
    
    def _adjust_neuron_positions(self, old_R, new_R):
        """调整神经元位置（按比例缩放）"""
        if old_R == 0:
            return
        
        ratio = new_R / old_R
        
        for neuron in self.brain.neurons:
            neuron.position = neuron.position * ratio
        
        print(f"  → 所有神经元位置已按比例调整: {ratio:.2f}x")
    
    def _record_adjustment(self, action, old_R, new_R):
        """记录调整历史"""
        self.adjustment_history.append({
            'step': self.brain.current_step,
            'action': action,
            'old_R': old_R,
            'new_R': new_R
        })
    
    def get_density(self):
        """获取空间密度（神经元/体积）"""
        volume = (4/3) * math.pi * (self.R ** 3)
        density = len(self.brain.neurons) / volume
        return density
    
    def get_status(self):
        """获取空间状态"""
        return {
            'radius': self.R,
            'neurons': len(self.brain.neurons),
            'density': self.get_density(),
            'adjustments': len(self.adjustment_history)
        }
```

```python
class MacOptimizedVisualizerWithAdjuster:

        elif event.key == ']':
            self.space_adjuster.expand(1.1)
        
        elif event.key == '{':
            self.space_adjuster.quick_shrink()
        
        elif event.key == '}':
            self.space_adjuster.quick_expand()
        
        elif event.key == 's':
            self.brain.storage.save_snapshot_force(self.brain)
            print("💾 已保存")

    def draw(self):

        if not self.should_update():
            return

        self.brain.R = self.space_adjuster.R

            return True
        return False
```
