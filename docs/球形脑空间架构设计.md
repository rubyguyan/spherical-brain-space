# 球形脑空间架构设计 v0.0.1

## 概述

本文档描述一个基于果蝇大脑结构的虚拟"球形脑空间"（电子培养皿）设计，用于模拟神经元的生存、连接、演化和信息处理。

---

## 1. 核心概念

### 什么是球形脑空间？

```
球形脑空间 = 电子培养皿 + 神经元生态系统

特点:
├── 3D 球形空间
├── 神经元可以"活着"
├── 有食物/能量系统
├── 有时间系统（可调节）
├── 可以接收外部输入（汉字等）
├── 可以 3D 可视化
├── 所有状态可保存
└── 神经元可起中文名字
```

## 2. 食物/能量系统

### 饱腹度等级

| 等级 | 饱腹度 | 获取方式 | 说明 |
|------|--------|----------|------|
| **低保** | 10% | 默认提供 | 10次吃饱，最基础的食物 |
| **1级** | 99% | 特殊触发 | 1次吃饱 |
| **2级** | 50% | 特殊触发 | 2次吃饱 |
| **3级** | 33% | 特殊触发 | 3次吃饱 |
| **4级** | 25% | 特殊触发 | 4次吃饱 |
| **5级** | 20% | 特殊触发 | 5次吃饱 |
| **信息处理** | 2% | 处理输入信息 | 神经元自己判断 |
| **输出奖励** | 2% | 对外输出 | 任何输出都奖励 |

### 食物提供规则

```
默认食物 (低保):
├── 每次提供 10% 饱腹度
├── 需要 10 次才能吃饱
└── 系统自动提供，保证神经元不会饿死

特殊食物触发方式:
├── 1级 (99%): 完成特定任务 / 输入特定内容
├── 2级 (50%): 与其他神经元形成新连接
├── 3级 (33%): 进入特定区域
├── 4级 (25%): 处理外部输入（如汉字）
├── 5级 (20%): 随机事件
├── 信息处理 (2%): 神经元处理输入信息后自己判断
└── 输出奖励 (2%): 神经元对外输出任何内容
```

### 神经元不会因饥饿死亡

```
重要规则: 神经元永远不会因饥饿死亡！

饥饿后果:
├── 饱腹度低 → 移动速度降低
├── 饱腹度低 → 分裂倾向增加
└── 饱腹度低 → 寻找食物的欲望增强

但不会死亡！
```

## 4. 三层球形空间结构

### 空间分层设计

```
球形脑空间 = 三层球体结构 (类似地球分层)

┌─────────────────────────────────────────────────────────┐
│                      大球 (外层)                        │
│   半径: R (可调整)                                      │
│   ┌─────────────────────────────────────────────────┐  │
│   │              中球 (中层)                         │  │
│   │   半径: R/2 (引力终点)                           │  │
│   │   ┌─────────────────────────────────────────┐   │  │
│   │   │          小球 (核心)                     │   │  │
│   │   │   半径: R/4 (暂不投放神经元)             │   │  │
│   │   │                                          │   │  │
│   │   │            ● 中心点                      │   │  │
│   │   │                                          │   │  │
│   │   └─────────────────────────────────────────┘   │  │
│   │                                                  │  │
│   └──────────────────────────────────────────────────┘  │
│                                                          │
└──────────────────────────────────────────────────────────┘

三层结构:
├── 小球 (核心层): 半径 R/4，暂不主动投放神经元
├── 中球 (内层): 半径 R/2，引力终点，可投放神经元
└── 大球 (外层): 半径 R，主要活动区域，可投放神经元
```

### 空间参数设置

```
默认参数 (可调整):
├── 大球半径 R = 1000 单位
├── 中球半径 = R/2 = 500 单位
├── 小球半径 = R/4 = 250 单位
└── 所有参数可在界面中调整

空间大小调整:
├── 调整大球半径 R
├── 中球半径自动 = R/2
├── 小球半径自动 = R/4
└── 神经元位置自动重新计算
```

### 神经元投放位置

```
投放区域:
├── 小球内部 (0 ~ R/4): 暂不主动投放
├── 中球区域 (R/4 ~ R/2): 可投放
├── 大球区域 (R/2 ~ R): 可投放 (主要区域)
└── 默认投放位置: R/2 ~ R 之间

投放位置选择:
┌─────────────────────────────────────┐
│  选择投放区域:                       │
│  ├── ○ 外层区域 (R/2 ~ R) [推荐]    │
│  ├── ○ 内层区域 (R/4 ~ R/2)         │
│  ├── ○ 核心区域 (0 ~ R/4) [不推荐]  │
│  └── ○ 随机位置                     │
└─────────────────────────────────────┘
```

### 引力终点机制

```
引力终点 = 中球外壁 (半径 R/2)

引力规则:
├── 神经元在大球区域 → 受引力向中心移动
├── 神经元到达中球外壁 → 引力停止
├── 神经元不会被动进入小球区域
└── 保护不动的神经元不被被动连接

为什么设置引力终点:
├── 防止神经元聚集到中心点
├── 保护"懒惰"神经元不被被动连接
├── 保持空间分布均匀
└── 让神经元有足够的活动空间
```

### 引力计算 (更新)

```
引力计算公式:
├── 距离 > R/2 (中球外): 引力正常作用
├── 距离 = R/2 (中球壁): 引力停止
├── 距离 < R/2 (中球内): 无引力
└── 距离 < R/4 (小球内): 无引力

实际速度计算:
if distance > R/2:
    actual_speed = v × (1 + g × cos(θ))
elif distance == R/2:
    actual_speed = v  # 引力停止
else:
    actual_speed = v  # 无引力影响
```


### 引力与神经元行为 (更新)

```
神经元在三层空间中的行为:
├── 外层区域 (R/2 ~ R)
│   ├── 受引力向中心移动
│   ├── 饱腹度高 → 可能抵抗引力
│   └── 饱腹度低 → 顺引力移动
├── 中球壁 (R/2)
│   ├── 引力停止
│   ├── 可以自由移动
│   └── 不会被拉向中心
├── 内层区域 (R/4 ~ R/2)
│   ├── 无引力影响
│   ├── 自由移动
│   └── 可投放神经元
└── 核心区域 (0 ~ R/4)
    ├── 无引力影响
    ├── 暂不主动投放神经元
    └── 保留给未来功能
```

## 5. 分裂规则

### 分裂条件

```
分裂触发条件（满足任一）:
├── 吃饱后 33 个时间步未连接
└── 半饱后 66 个时间步未连接

注意: 吃饱和半饱满足任一即可
```

### 分裂过程

```
时间步 0: 神经元A (饱腹度 100%, 连接数 0)
    ↓
时间步 1-33: 未形成连接
    ↓
时间步 33: 触发分裂!
    ↓
时间步 34: 
    ├── 神经元A' (克隆体, 饱腹度 50%, 新位置)
    └── 神经元A (消亡)
```

### 连接重置

```
当神经元形成新连接时:
├── 重置吃饱计数器为 0
├── 重置半饱计数器为 0
└── 开始新的生命周期
```

## 5. 移动规则

### 移动速度

| 对象 | 速度 | 说明 |
|------|------|------|
| **神经元** | v (基础速度) | 可调节 |
| **突触** | v/10 | 突触跟随神经元移动 |

### 移动行为

```
神经元移动倾向:
├── 饱腹度 < 26% → 向能量源移动 (触发阈值改为26%)
├── 连接数 = 0 → 向其他神经元移动
├── 有特定目标 → 向目标移动
└── 无目标 → 随机游走

神经元自由度:
├── 可以呆在原地不动
├── 可以来回跑动
├── 可以进入任何区域
└── 完全自由！
```

### 神经元群组移动

```
当多个神经元连接形成群组时:
├── 群组视为一个整体
├── 整体移动速度增加 2%
├── 连接越多，协作越强
└── 群组内神经元共享部分状态
```

## 6. 突触系统

### 突触结构

```
突触 = 神经元之间的连接点

结构:
┌────────────┐
│ 神经元A末端 │ ← 突触前膜
├────────────┤
│  突触间隙   │ ← 弹性: 0 ~ 120纳米 (可伸缩5倍)
├────────────┤
│ 神经元B末端 │ ← 突触后膜
└────────────┘

突触间隙弹性:
├── 最短: 0 纳米 (完全接触)
├── 标准: 20 纳米
├── 最长: 120 纳米 (伸长5倍)
└── 可动态调节
```

### 突触自主行为

```
突触可以:
├── 自己长出 - 神经元主动伸出突触
├── 自己长出第二个 - 神经元可以长出第2个突触
├── 自己长出第三个 - 神经元最多可以有3个突触
├── 自己收回 - 神经元收回突触
├── 自己丢弃 - 神经元断开连接
└── 弹性伸缩 - 间隙可调节

突触数量限制:
├── 最少: 0 个
├── 最多: 3 个
└── 由神经元自己判断是否需要长出新突触
```

### 突触间隙弹性

| 状态 | 间隙大小 | 说明 |
|------|----------|------|
| **紧密** | 0 纳米 | 完全接触 |
| **标准** | 20 纳米 | 正常状态 |
| **拉伸** | 120 纳米 | 最大伸展 |

## 7. 神经元信息传递系统

### 连接神经元的信息交换

```
当两个神经元连接后:
├── 自动开启双向信息通道
├── 双方可以互相发送信息
├── 信息传递通过突触进行
└── 神经元自己决定发送什么信息
```

### 信息传递类型

| 类型 | 方向 | 可见性 | 收集 |
|------|------|--------|------|
| **内部传递** | 神经元↔神经元 | 不可见 | 不收集 |
| **对外输出** | 神经元→外部 | 可见 | 收集 |
| **接收输入** | 外部→神经元 | 可见 | 收集 |

### 信息传递规则

```
内部传递 (神经元之间):
├── 两个连接的神经元自动交换信息
├── 信息内容神经元自己决定
├── 传递速度受突触间隙影响
├── 间隙越小传递越快
├── 看不见 → 不收集
└── 用于神经元内部协调

对外输出:
├── 神经元向外部系统输出信息
├── 输出内容神经元自己决定
├── 看得见 → 收集保存
├── 用于分析神经元行为
└── 输出奖励 2% 饱腹度

接收输入:
├── 外部系统向神经元输入信息
├── 如: 汉字、词语、文本
├── 神经元处理输入后自己判断
├── 看得见 → 收集保存
└── 处理奖励 2% 饱腹度
```

### 信息传递记录结构

```
信息记录:
├── 内部传递记录 (可选)
│   ├── 时间戳
│   ├── 发送神经元ID
│   ├── 接收神经元ID
│   ├── 信息内容
│   └── 突触间隙大小
├── 对外输出记录
│   ├── 时间戳
│   ├── 神经元ID
│   ├── 输出内容
│   └── 触发原因
└── 接收输入记录
    ├── 时间戳
    ├── 输入内容
    ├── 接收神经元ID
    └── 处理结果
```

### 信息传递分析

```
分析内容:
├── 输出频率统计
├── 输出内容分析
├── 输入响应分析
├── 神经元间信息流
├── 信息传递效率
└── 群体决策分析
```

## 9. 神经元群体网络系统

### 群体形成规则

```
当神经元协作时:
├── 2个神经元连接 → 形成协作关系
├── 3个及以上神经元连接 → 形成群体网络
├── 群体网络 = 新的"超级神经元"
└── 有统一的输入/输出接口
```

### 群体网络发现机制

```
发现方式: 程序主动检测 (非神经元主动通知)

检测流程:
┌─────────────────────────────────────┐
│  每个时间步自动检测                  │
│  ├── 扫描所有神经元连接状态         │
│  ├── 使用广度优先搜索 (BFS)         │
│  └── 找出所有连接组                 │
└──────────────┬──────────────────────┘
               ▼
┌─────────────────────────────────────┐
│  判断连接组大小                      │
│  ├── < 3 个 → 不形成群体网络        │
│  └── ≥ 3 个 → 形成群体网络          │
└──────────────┬──────────────────────┘
               ▼
┌─────────────────────────────────────┐
│  自动创建群体网络                    │
│  ├── 分配网络ID                     │
│  ├── 创建集群文件夹                 │
│  ├── 复制神经元文件                 │
│  └── 通知用户                       │
└─────────────────────────────────────┘

为什么是程序主动检测:
├── 神经元只负责生存和连接
├── 神经元不知道"群体网络"概念
├── 程序负责管理和记录
└── 类似于观察者在培养皿中观察细胞
```

### 群体网络通知用户

```
通知时机:
├── 新群体网络形成时
├── 群体网络输出有意义信息时
├── 群体网络结构变化时
└── 群体网络达到里程碑时

通知方式:
┌─────────────────────────────────────┐
│  🔔 群体网络通知                     │
├─────────────────────────────────────┤
│  网络ID: network_001                │
│  成员数: 3 个神经元                 │
│  成员: 小明, 学习一号, 导航专家     │
│  状态: 新形成                       │
│  输出: "汝是否想调整规则" → "是"   │
│                                     │
│  [查看详情] [输入信息] [忽略]       │
└─────────────────────────────────────┘

有意义输出识别:
├── 包含中文词语
├── 包含数字
├── 包含逻辑表达
├── 重复出现的模式
└── 与输入相关的响应
```

### 群体网络特性

```
群体网络 (≥3个神经元):
├── 统一的输入接口 → 接收程序信息
├── 统一的输出接口 → 输出给程序
├── 内部神经元自由传递信息
├── 传递内容不限 (包括"乱码")
└── 形成独立的小型"大脑"
```

### 群体网络结构

```
群体网络结构:

    ┌─────────────────────────────────────┐
    │         外部程序接口                │
    │    ┌─────────┐    ┌─────────┐      │
    │    │ 输入接口 │    │ 输出接口 │      │
    │    └────┬────┘    └────┬────┘      │
    └─────────┼──────────────┼───────────┘
              │              │
              ▼              ▲
```

## 13. 突触数据

```
突触 = 神经元之间的连接点

结构:
┌────────────┐

├────────────┤

突触不是独立部件，而是两个神经元共同构成的连接点
```

## 14. 完整模拟器代码框架

```python
import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D
from dataclasses import dataclass, field
from typing import List, Optional, Dict
import random
import json

@dataclass
class Neuron:
    id: int
    source_file: str
    alias: str = ""  # 中文别名, 1-9个字
    position: np.ndarray = field(default_factory=lambda: np.zeros(3))
    satiety: float = 1.0
    speed: float = 5.0
    connections: int = 0
    full_count: int = 0
    half_count: int = 0
    neuron_type: str = ""
    group_id: int = 0  # 所属群组
    synapse_count: int = 0  # 当前突触数量 (最多3个)
    
    # 行为记录
    last_position: np.ndarray = field(default_factory=lambda: np.zeros(3))
    no_move_count: int = 0  # 不动计数
    output_count: int = 0  # 输出计数
    last_outputs: List[str] = field(default_factory=list)  # 最近输出
    
    def set_alias(self, name: str):
        if 1 <= len(name) <= 9:
            self.alias = name
    
    def process_input(self, info: str) -> Optional[str]:
        response = None
        if random.random() > 0.3:
            response = f"[{self.alias}] 处理: {info}"
            self.satiety = min(1.0, self.satiety + 0.02)
        return response
    
    def answer_question(self, question: str) -> Optional[str]:
        answers = [
            "同意调整", "保持现状", "需要思考", "暂不调整",
            "调整部分规则", "完全同意", "部分同意",

    def move(self, target=None, group_bonus=0, gravity_factor=0.03):
        self.last_position = self.position.copy()
        actual_speed = self.speed * (1 + group_bonus)
        
        if target is not None:
            direction = target - self.position
            if np.linalg.norm(direction) > 0:
                direction = direction / np.linalg.norm(direction)
        else:
            direction = np.random.randn(3)
            direction = direction / np.linalg.norm(direction)
        
        self.position += direction * actual_speed
    
    def move_with_gravity(self, center=np.zeros(3), gravity_factor=0.03, target=None, group_bonus=0):
        self.last_position = self.position.copy()
        base_speed = self.speed * (1 + group_bonus)
        
        if target is not None:
            direction = target - self.position
        else:
            direction = np.random.randn(3)
        
        if np.linalg.norm(direction) > 0:
            direction = direction / np.linalg.norm(direction)
        
        to_center = center - self.position
        if np.linalg.norm(to_center) > 0:
            to_center = to_center / np.linalg.norm(to_center)
        
        cos_angle = np.dot(direction, to_center)
        gravity_effect = gravity_factor * cos_angle
        actual_speed = base_speed * (1 + gravity_effect)
        
        self.position += direction * actual_speed
        return actual_speed

    def update(self):

        if self.connections == 0:
            self.full_count += 1

        # 检测是否不动
        move_dist = np.linalg.norm(self.position - self.last_position)

@dataclass
class Synapse:
    pre_neuron: Neuron
    post_neuron: Neuron
    gap: float = 20.0
    strength: float = 0.5
    
    def extend_gap(self, amount: float):
        self.gap = min(120, self.gap + amount)
    
    def shrink_gap(self, amount: float):
        self.gap = max(0, self.gap - amount)
    
    def remove(self):
        self.pre_neuron.connections -= 1
        self.post_neuron.connections -= 1
    
    def transfer_speed(self) -> float:
        return 1.0 / (1 + self.gap / 20.0)
    
    def internal_transfer(self, message: str, from_pre: bool = True) -> Dict:
        sender = self.pre_neuron if from_pre else self.post_neuron
        receiver = self.post_neuron if from_pre else self.pre_neuron
        return {
            'time': 0,
            'from_id': sender.id,
            'to_id': receiver.id,
            'message': message,
            'gap': self.gap,
            'speed': self.transfer_speed(),
            'visible': False
        }

    def generate_deploy_position(self, layer: str = "outer") -> np.ndarray:
        if layer == "inner":
            r = np.random.uniform(self.inner_radius * 0.5, self.inner_radius)
        elif layer == "middle":
            r = np.random.uniform(self.inner_radius, self.middle_radius)
        else:
            r = np.random.uniform(self.middle_radius, self.outer_radius)
        
        theta = np.random.uniform(0, 2 * np.pi)
        phi = np.random.uniform(0, np.pi)
        
        x = r * np.sin(phi) * np.cos(theta)
        y = r * np.sin(phi) * np.sin(theta)
        z = r * np.cos(phi)
        
        return np.array([x, y, z])
    
    def move_with_gravity(self, neuron: Neuron, target=None, group_bonus=0):
        distance = np.linalg.norm(neuron.position - self.center)
        
        neuron.last_position = neuron.position.copy()
        base_speed = neuron.speed * (1 + group_bonus)
        
        if target is not None:
            direction = target - neuron.position
        else:
            direction = np.random.randn(3)
        
        if np.linalg.norm(direction) > 0:
            direction = direction / np.linalg.norm(direction)
        
        if distance > self.middle_radius:
            to_center = self.center - neuron.position
            if np.linalg.norm(to_center) > 0:
                to_center = to_center / np.linalg.norm(to_center)
            
            cos_angle = np.dot(direction, to_center)
            gravity_effect = self.GRAVITY_FACTOR * cos_angle
            actual_speed = base_speed * (1 + gravity_effect)
        else:
            actual_speed = base_speed
        
        neuron.position += direction * actual_speed
        return actual_speed

    def check_meaningful_output(self, output: str) -> bool:
        import re
        if re.search(r'[\u4e00-\u9fff]+', output):
            return True
        if re.search(r'\d+', output):
            return True
        if re.search(r'(是|否|好|不|对|错|想|要)', output):
            return True
        if len(output) > 3 and output.isalpha():
            return True
        return False
    
    def find_connected_group(self, neuron: Neuron) -> List[Neuron]:
        group = [neuron]
        visited = {neuron.id}
        queue = [neuron]
        
        while queue:
            current = queue.pop(0)
            for s in self.synapses:
                other = None
                if s.pre_neuron.id == current.id:
                    other = s.post_neuron
                elif s.post_neuron.id == current.id:
                    other = s.pre_neuron
                
                if other and other.id not in visited:
                    visited.add(other.id)
                    group.append(other)
                    queue.append(other)
        
        return group

    def step(self):
        if self.paused:
            return
        self.time_step += 1
        self.alerts = []
        
        for neuron in self.neurons:
            neuron.update()
            
            neuron.move_with_gravity(
                center=self.center,
                gravity_factor=self.GRAVITY_FACTOR,
                group_bonus=0.02 if neuron.connections > 0 else 0
            )
            
            if neuron.satiety < 0.1:
                self.provide_food(neuron, 10)
            if neuron.should_split():
                self.split_neuron(neuron)
            
            answer = neuron.answer_question(self.FIXED_QUESTION)
            if answer:
                self.answers.append({
                    'time': self.time_step,
                    'neuron_id': neuron.id,
                    'question': self.FIXED_QUESTION,
                    'answer': answer
                })
            
            output = neuron.output()
            if output:
                self.outputs.append({
                    'time': self.time_step,
                    'neuron_id': neuron.id,
                    'content': output
                })
                if output not in self.unique_outputs:
                    self.unique_outputs[output] = {
                        'first_time': self.time_step,
                        'last_time': self.time_step,
                        'count': 1,
                        'neuron_id': neuron.id
                    }
                else:
                    self.unique_outputs[output]['count'] += 1
                    self.unique_outputs[output]['last_time'] = self.time_step
            
            if neuron.is_not_moving():
                if neuron.id not in self.not_moving_neurons:
                    self.not_moving_neurons.append(neuron.id)
                    self.alerts.append({
                        'type': 'not_moving',
                        'neuron_id': neuron.id,
                        'alias': neuron.alias,
                        'message': f"神经元 {neuron.alias} 已连续 100 步未移动"
                    })
            
            if neuron.is_high_output():
                if neuron.id not in self.high_output_neurons:
                    self.high_output_neurons.append(neuron.id)
                    self.alerts.append({
                        'type': 'high_output',
                        'neuron_id': neuron.id,
                        'alias': neuron.alias,
                        'message': f"神经元 {neuron.alias} 高频输出中"
                    })
            
            if self.check_meaningful_output(output):
                self.alerts.append({
                    'type': 'meaningful_output',
                    'neuron_id': neuron.id,
                    'alias': neuron.alias,
                    'content': output,
                    'message': f"神经元 {neuron.alias} 输出有意义信息: {output}"
                })
        
        self.process_internal_messages()
        self.check_connections()
        self.update_groups()
        
        network_alerts = self.check_networks()
        self.alerts.extend(network_alerts)
        
        if self.time_step - self.last_save_time >= self.AUTO_SAVE_INTERVAL:
            self.auto_save()
            self.last_save_time = self.time_step

    def command(self, cmd: str) -> Optional[Dict]:
        """用户指令系统"""
        parts = cmd.split()
        action = parts[0] if parts else ""
        
        if action == "pause":
            self.paused = True
            return {'status': 'paused'}
        
        elif action == "resume":
            self.paused = False
            return {'status': 'resumed'}
        
        elif action == "set_time":
            ratio = int(cmd.split("ratio=")[1])
            self.time_ratio = ratio
            return {'status': 'time_set', 'ratio': ratio}
        
        elif action == "analyze":
            return self.analyze_outputs()
        
        elif action == "transfer_output":
            from_id = int(cmd.split("from=")[1].split()[0])
            to_id = int(cmd.split("to=")[1])
            from_n = next((n for n in self.neurons if n.id == from_id), None)
            to_n = next((n for n in self.neurons if n.id == to_id), None)
            if from_n and to_n and from_n.last_outputs:
                content = from_n.last_outputs[-1]
                response = to_n.process_input(content)
                return {'status': 'transferred', 'content': content, 'response': response}
            return {'status': 'failed'}
        
        elif action == "rename":
            nid = int(cmd.split("id=")[1].split()[0])
            new_alias = cmd.split("new_alias=")[1]
            neuron = next((n for n in self.neurons if n.id == nid), None)
            if neuron:
                neuron.set_alias(new_alias)
                return {'status': 'renamed', 'new_alias': new_alias}
            return {'status': 'failed'}
        
        elif action == "add_neuron":
            nid = int(cmd.split("id=")[1].split()[0])
            alias = cmd.split("alias=")[1] if "alias=" in cmd else f"神经元{nid}"
            neuron = Neuron(
                id=nid,
                source_file=f"manual_{nid}.swc",
                alias=alias,
                position=np.random.randn(3) * self.radius * 0.5
            )
            self.add_neuron(neuron)
            return {'status': 'added', 'neuron_id': nid, 'alias': alias}
        
        elif action == "status":
            return {
                'time_step': self.time_step,
                'time_ratio': self.time_ratio,
                'paused': self.paused,
                'neuron_count': len(self.neurons),
                'synapse_count': len(self.synapses),
                'output_count': len(self.outputs),
                'alerts': self.alerts
            }
        
        return {'status': 'unknown_command'}

    def send_signal_to_neuron(self, neuron_id: int, signal: str) -> Dict:

    def create_cluster_folder(self, network_id: str):
        import os
        import shutil
        
        cluster_dir = f"./集群_{network_id}"
        if not os.path.exists(cluster_dir):
            os.makedirs(cluster_dir)
        
        network = next((n for n in self.networks if n['id'] == network_id), None)
        if network:
            for member_id in network['members']:
                neuron = next((n for n in self.neurons if n.id == member_id), None)
                if neuron:
                    src_file = neuron.source_file
                    if os.path.exists(src_file):
                        new_name = f"神经元_{neuron.alias}_id{neuron.id}.swc"
                        dst_file = os.path.join(cluster_dir, new_name)
                        shutil.copy2(src_file, dst_file)
                        neuron.source_file = dst_file
        
        return cluster_dir
    
    def mutate_neuron_in_cluster(self, network_id: str, neuron_id: int, mutation_type: str = "random"):
        import os
        import random
        from datetime import datetime
        
        network = next((n for n in self.networks if n['id'] == network_id), None)
        if not network or neuron_id not in network['members']:
            return None
        
        neuron = next((n for n in self.neurons if n.id == neuron_id), None)
        if not neuron:
            return None
        
        cluster_dir = f"./集群_{network_id}"
        if not os.path.exists(cluster_dir):
            return None
        
        original_file = neuron.source_file
        if not os.path.exists(original_file):
            return None
        
        timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
        new_file = os.path.join(cluster_dir, f"神经元_{neuron.alias}_变异_{timestamp}.swc")
        
        with open(original_file, 'r') as f:
            lines = f.readlines()
        
        new_lines = []
        for line in lines:
            if line.startswith('#'):
                new_lines.append(line)
            else:
                parts = line.strip().split()
                if len(parts) >= 7:
                    if mutation_type == "random":
                        parts[2] = str(float(parts[2]) + random.uniform(-1, 1))
                        parts[3] = str(float(parts[3]) + random.uniform(-1, 1))
                        parts[4] = str(float(parts[4]) + random.uniform(-1, 1))
                    new_lines.append(' '.join(parts) + '\n')
        
        with open(new_file, 'w') as f:
            f.writelines(new_lines)
        
        neuron.source_file = new_file
        return new_file

            return True

            network['members'].remove(neuron_id)

            return True
        return False
```
